## Quick Start

For immediate setup, here are the most common installation commands:

### JavaScript/TypeScript
```bash copy
npm install glin-profanity
```

### Python
```bash copy
pip install glin-profanity
```

### Verification
```javascript copy title="test.js"
import { checkProfanity } from 'glin-profanity';
console.log('âœ“ Ready:', checkProfanity('test').containsProfanity === false);
```

```python copy title="test.py"
from glin_profanity import Filter
filter = Filter()
print('âœ“ Ready:', not filter.check_profanity('test')['contains_profanity'])
```

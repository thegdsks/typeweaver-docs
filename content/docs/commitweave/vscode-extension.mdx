Bring the power of AI-driven commit messages directly into your VS Code editor with the CommitWeave extension. Generate conventional commits, validate messages, and streamline your Git workflow without leaving your IDE.

## Installation

### From VS Code Marketplace

The easiest way to get started with CommitWeave in VS Code:

```bash tab="Command Palette"
# Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)
# Type: "Extensions: Install Extensions"
# Search for "CommitWeave"
# Click Install
```

```bash tab="VS Code CLI"
# Install via VS Code command line
code --install-extension commitweave.commitweave-vscode

# Verify installation
code --list-extensions | grep commitweave
```

```bash tab="Extensions View"
# Open Extensions view (Ctrl+Shift+X / Cmd+Shift+X)
# Search: "CommitWeave"  
# Publisher: CommitWeave Team
# Click Install button
```

### Local Development Install

For development or pre-release versions:

```bash tab="From VSIX"
# Download .vsix file from releases
# Install via Command Palette
# "Extensions: Install from VSIX..."
# Select downloaded commitweave-x.x.x.vsix file
```

```bash tab="Build from Source"
# Clone the repository
git clone https://github.com/GLINCKER/commitweave-vscode
cd commitweave-vscode

# Install dependencies
npm install

# Package extension
npm run package

# Install locally
code --install-extension commitweave-*.vsix
```

### System Requirements

- **VS Code**: Version 1.74.0 or higher
- **Node.js**: Version 16+ (for CLI backend)
- **Git**: Must be available in PATH
- **CommitWeave CLI**: Auto-installed or manual setup

<Callout type="info">
The extension will automatically install CommitWeave CLI if not detected. You can also install it manually with `npm install -g commitweave`.
</Callout>

## Commands

### Create Commit

Generate commit messages interactively with full customization.

**Command:** `CommitWeave: Create Commit`  
**Shortcut:** `Ctrl+Shift+G C` (Windows/Linux) / `Cmd+Shift+G C` (macOS)

<ImageZoom
  src="/images/vscode/create-commit-command.png" 
  alt="CommitWeave Create Commit command in VS Code Command Palette"
  width={800}
  height={400}
/>

**Usage:**
1. Stage your changes in Source Control view
2. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
3. Type "CommitWeave: Create Commit"
4. Follow the interactive prompts:
   - Select commit type (feat, fix, docs, etc.)
   - Enter scope (optional)
   - Provide description
   - Add body text (optional)
   - Mark breaking changes

**Example Flow:**
```
? Select commit type: › feat
? Enter scope (optional): › auth  
? Enter description: › add OAuth2 integration
? Enter body (optional): › 
  Implement secure third-party authentication with
  automatic token refresh and proper error handling
? Breaking change? › No
? Related issues: › Closes #123

Generated: feat(auth): add OAuth2 integration
```

**Shortcut:** `Ctrl+Shift+G A` (Windows/Linux) / `Cmd+Shift+G A` (macOS)

**Features:**

- **Context Awareness**: Understands code patterns and intent  
- **Conventional Format**: Generates properly formatted commits
- **Multiple Suggestions**: Provides alternative commit messages
- **Confidence Scoring**: Shows AI confidence in suggestions

**Usage:**

## Advanced Features

### Workspace Configuration

Set up project-specific CommitWeave settings:

```json title=".vscode/settings.json"
{
  "commitweave.ai.provider": "openai",
  "commitweave.ai.model": "gpt-4-turbo", 
  "commitweave.commit.requireScope": true,
  "commitweave.commit.allowedScopes": [
    "frontend",
    "backend", 
    "api",
    "ui",
    "docs"
  ],
  "commitweave.ui.autoCommit": false,
  "commitweave.ui.showNotifications": true
}
```

## Troubleshooting

### Extension Not Working

**Check Prerequisites:**
1. **CommitWeave CLI installed**: Run `commitweave --version` in terminal
2. **Git repository**: Ensure you're in a Git-initialized project
3. **API Keys configured**: Set `OPENAI_API_KEY` or `CLAUDE_API_KEY`
4. **VS Code version**: Update to latest stable version

### AI Features Unavailable

**Common Solutions:**
```bash
# Verify API key
echo $OPENAI_API_KEY

# Test CLI directly  
commitweave ai --dry-run

# Check extension logs
# View > Output > CommitWeave Extension
```

### Performance Issues

**Optimization Tips:**
- **Reduce max tokens**: Lower to 50-100 for faster responses
- **Use GPT-3.5**: Switch from GPT-4 for speed
- **Enable caching**: Cache similar commit patterns
- **Exclude large files**: Add to `.commitweaveignore`

Customize shortcuts in `File > Preferences > Keyboard Shortcuts`:

```json title="keybindings.json"
[
  {
    "key": "ctrl+alt+c",
    "command": "commitweave.aiCommit",
    "when": "scmProvider == 'git'"
  },
  {
    "key": "ctrl+alt+q", 
    "command": "commitweave.quickCommit",
    "when": "scmProvider == 'git'"
  }
```

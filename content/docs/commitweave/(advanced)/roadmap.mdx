## Overview

CommitWeave has evolved significantly with multiple shipped features, active development, and exciting plans ahead. This roadmap shows our current state and future direction.

## Development Status

### <CheckCircle className="w-5 h-5 text-green-600 dark:text-green-400 inline mr-2" />Shipped Features

We've successfully shipped **11 major features** including:
- Interactive CLI with 23ms cold-start performance
- Full VS Code extension with 5 commands  
- Multi-AI provider support (OpenAI, Claude, Mock)
- Advanced configuration system with export/import
- Comprehensive testing suite with cross-platform support

### <Clock className="w-5 h-5 text-blue-600 dark:text-blue-400 inline mr-2" />Currently Building

**2 active development** items:
- Git hooks system with pre/post-commit templates
- Plugin system for custom commit types and extensibility

### <Target className="w-5 h-5 text-orange-600 dark:text-orange-400 inline mr-2" />Planned Next

**6 planned features** for upcoming releases:
- Multi-repository support
- Web-based configuration builder
- Enhanced AI providers (Google PaLM, local models)
- Analytics & usage metrics
- Issue tracker integrations
- JetBrains IDEs extension

### <Lightbulb className="w-5 h-5 text-purple-600 dark:text-purple-400 inline mr-2" />Future Ideas

**6 potential features** we're exploring:
- Framework configuration templates
- Configuration backup & sync
- Browser extensions for web Git platforms
- Mobile app for repository management
